Informações Gerais
Este projeto é um jogo de plataforma 2D desenvolvido em C++ utilizando programação orientada a objetos (OO). O jogo foi criado como parte dos estudos nas disciplinas Técnicas de Programação e Conceitos da Engenharia de Software. Ele foi desenvolvido para ajudar na compreensão dos conceitos abordados nessa matéria.

Sistema Operacional
O jogo foi desenvolvido e testado no seguinte sistema operacional:

Windows
Compilador
O código foi compilado usando o seguinte compilador:

GCC (GNU Compiler Collection) versão 6.3.0
Bibliotecas Utilizadas
O jogo utiliza as seguintes bibliotecas:

SFML (Simple and Fast Multimedia Library) versão 2.6.1: Utilizada para renderização gráfica, manipulação de eventos e sons.
STL (Standard Template Library): Utilizada para estruturas de dados e algoritmos padrão da linguagem C++.

Funcionalidades do Jogo
Orientação a Objetos: O jogo é estruturado seguindo os princípios da Programação Orientada a Objetos (POO).
Menus Gráficos: Inclui menus para navegação, como:
Tela de colocação do jogador
Seleção de fases
Opção para fechar o aplicativo
Duas Fases Distintas: O jogo apresenta duas fases, cada uma com:
Chefões diferentes
Inimigos e obstáculos distintos
Requisitos para Execução
Sistema Operacional: Windows
Compilador: GCC 6.3.0 ou superior
Biblioteca Gráfica: SFML 2.6.1
IDE Recomendada: Visual Studio 2022 ou qualquer outra IDE que suporte C++ e permita a configuração dos diretórios de inclusão e bibliotecas adicionais.

Instruções para Compilação e Execução
Baixe o Projeto:

Certifique-se de que a pasta do projeto contenha a pasta SFML-2.6.1, que já inclui os diretórios include e lib necessários para o SFML.
Configuração do Projeto na IDE:

Abra o projeto em uma IDE compatível (recomenda-se o Visual Studio 2022).
Configure os Diretórios de Inclusão Adicionais para apontar para a pasta SFML-2.6.1\include dentro do diretório do projeto.
Configure os Diretórios de Bibliotecas Adicionais para apontar para a pasta SFML-2.6.1\lib.
Adicione as bibliotecas necessárias (sfml-graphics.lib, sfml-window.lib, sfml-system.lib) ao seu projeto.
Compilação:

Compile o projeto através da IDE. Certifique-se de que a configuração está definida como Release ou Debug, conforme necessário.
Execução:

Após a compilação bem-sucedida, execute o jogo a partir da IDE.
